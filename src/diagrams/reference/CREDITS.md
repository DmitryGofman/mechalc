These diagrams were sourced from **Wikimedia Commons** as higher-detail references
alongside the project's own hand-built SVGs in `../`. They are kept separate so the
licensing is explicit and so you can decide per-diagram whether to ship them.

> (2) **ShareAlike** — any modified version of *that image* must also be CC BY-SA.
> This does *not* "infect" your source code, but it is a real obligation for the
> images themselves. The ISO thread diagram is **Public Domain** (no obligation).
